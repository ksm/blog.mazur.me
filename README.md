# Setup

```
bundle install
```

# Usage

```
bundle exec jekyll serve
```

Use `-l` for live reload. Use `--host=0.0.0.0` for serving over LAN.

- https://jekyllrb.com/docs/configuration/
- https://jekyllrb.com/docs/usage/
